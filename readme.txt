DESCRIPTION:

Krib is a modern, student-focused app landing page template designed to showcase apps that help students find campus housing. It includes banner areas, call-to-action buttons, video support, testimonial sliders, pricing sections, and a clean, responsive layout that presents your app professionally. Krib is built with organized, easy-to-customize code and is mobile- and retina-ready.
